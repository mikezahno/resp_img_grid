#Responsive-Image-Grid-Formatter

####Drupal 7.x Module

### Features
* Format any image with responsive markup
* Touch-Friendly
* Optional: Set the box (square background) to the dominant color of the image (needs Color Thief lib)
* Integrates with Libraries 2.x
